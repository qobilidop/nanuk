# M3: MAP Language Stack Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** MAP programs flow through the whole language stack — eDSL → MapProgram IR → lowering → map asm — with validator + step-exact interpreter, demo-program parity against the hand-written assembly, and playground support.

**Architecture & interfaces:** frozen in [the M3 design doc](../specs/2026-07-11-m3-map-lang-ir-design.md) — read it first; every message/field/function name there is normative. Parser `Program` and its consumers are untouched.

**Tech Stack:** protobuf (protoc in devcontainer), Python (uv workspaces: compiler/, lang/, web/py), Vite/Svelte playground, Pyodide wheels.

## Global Constraints

- Parser IR path untouched: all existing compiler/lang/web tests must stay green unmodified.
- interp_map's step accounting mirrors lower_map instruction-for-instruction (steps parity is asserted, budget exhaustion included).
- scapy never enters wheels (playground bundle rule).
- Verify in devcontainer; commit per task; push + watch CI at the end.

### Task 1: proto + validate_map (compiler/)
MapProgram/MapOp/... messages per design doc; regen `nanuk_ir_pb2.py` (find the regen command in compiler/README or pyproject scripts; else `protoc --python_out`). `validate_map`: value discipline (define-before-use, unique ids, no cross-state), state/table refs (incl. Lookup.miss_state), ranges (offset ±512, nbytes 1-8, imm 16-bit signed for MapAdd / unsigned for MapConst, hdr_id ≤ 15, field ≤ 15, delta range, table_id < 4, widths ≤ 64). Tests: minimal valid program passes; each violation raises.

### Task 2: lower_map (compiler/)
`lower_map.to_map_asm(MapProgram, *, check=True) -> str`: r0-r2 per-state linear allocation, r3 scratch for dispatch constants; ops → `ld/ldmd/movi/addi/st/csumupd/lookup` lines; terminators → `send/drop/jmp` + BEQ chains. Store of a Const value materializes via its MOVI (Const is an op that allocates). Tests: golden asm text for an l2fwd-shaped program; out-of-registers raises with live-value message.

### Task 3: interp_map (compiler/)
`interp_map(program, packet, pp, tables, ingress)` → MapInterpResult(verdict/error/egress/delta/steps/frame). Window semantics identical to the Sail model (headroom 32, zero-filled, tail passthrough in result frame); cost model mirrors Task 2 exactly. Differential test (NANUK_COSIM): random + corpus packets through interp_map vs run_map(assembled lowering) — all fields equal, steps included.

### Task 4: eDSL MapProgram (lang/)
`nanuk_lang.map_program.MapProgram` (+ `MapStateCompiler`): decorator/build_ir/compile mirroring Parser; `mp.table`, `mp.header` binding, `s.load/load_md/const/add/store/lookup/csum_update/send/drop/dispatch/goto`; byte-alignment CompileError for non-byte fields; MD_* constants exported. Unit tests: IR snapshots for small programs, alignment errors, lookup miss-target wiring.

### Task 5: demo parity (lang/ + examples/)
l2fwd, ttl, tunnel push+pop as eDSL programs (`lang/nanuk_lang/programs/map_*.py`, `examples/*/fwd.py` with make_map()/build_ir()); tests (NANUK_COSIM): assembled-eDSL vs hand-written .asm behaviorally identical over the M1 corpus + tunnel frames through the golden emulator (full MapResult diff), and interp_map parity on the same corpus.

### Task 6: playground (web/)
bridge.py: `compile_map_source` (render MapProgram IR with line ranges + emission counts via the ordered-walk machinery) + `run_map_packet` (interp_map, preset table set baked per example); one MAP preset (l2fwd eDSL source) in the example chips; result panel MAP variant (egress bitmap, delta, frame hex); wheels rebuild; Node Pyodide test extended. UI copy: pane names unchanged (nanuk lang | IR | asm).

### Task 7: CI + docs
Full local CI-equivalent run; push; watch green. Lab notes (`guide/notes/2026-07-11-m3-map-lang-lab-notes.md`); memory update; MAT-arc status updates in the two design docs (M3 done → arc complete).
