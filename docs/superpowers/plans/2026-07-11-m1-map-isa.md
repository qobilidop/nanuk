## Frozen architectural constants (MAP plane)

**Parameters** (`spec/map-model/params.sail`): `headroom_bytes = 32`, `buf_bytes = 256` (window = 288), `imem_words = 1024`, `n_ports = 4`, `n_tables = 4`, `table_max_entries = 64`, `smd_in_slots = 8` (PP pass-through), `nhdr = 16`, `step_budget = 256`.

**Encodings** — 32-bit words, opcode [31:26], reserved bits zero-enforced (nonzero ⇒ ILLEGAL); `off` fields are 10-bit two's-complement; register field 3 bits (`rz` = 4):

| Op | Instruction | Layout below opcode |
|---|---|---|
| 0x01 | `LD rd, hdr, off, n` | rd[25:23] hdr[22:19] off[18:9] (n-1)[8:6] |
| 0x02 | `ST rs, hdr, off, n` | rs[25:23] hdr[22:19] off[18:9] (n-1)[8:6] |
| 0x03 | `LDMD rd, field` | rd[25:23] field[22:19] |
| 0x04 | `MOVI rd, imm16` | rd[25:23] imm[15:0] |
| 0x05 | `ADDI rd, rs, imm16` | rd[25:23] rs[22:20] imm[15:0] (imm sign-extended) |
| 0x06 | `BEQ rs, rt, tgt16` | rs[25:23] rt[22:20] tgt[15:0] |
| 0x07 | `BNE rs, rt, tgt16` | rs[25:23] rt[22:20] tgt[15:0] |
| 0x08 | `JMP tgt16` | tgt[15:0] |
| 0x09 | `LOOKUP rd, tbl, rs, tgt16` | rd[25:23] tbl[22:19] rs[18:16] tgt[15:0] |
| 0x0A | `CSUMUPD hdr, off` | hdr[25:22] off[21:12] |
| 0x0B | `SEND rs, delta` | rs[25:23] delta[22:13] (delta two's-complement) |
| 0x0C | `DROP` | (all reserved) |

**Addressing:** effective window index = `headroom_bytes + hdr_base(hdr) + off`, where `hdr_base(15) = 0` (**`h_frame`**, always valid) and `hdr_base(0..14) = unsigned(hdr_offset_in[h])`, valid only if `hdr_present_in[h]` (else `err_hdr_absent`). Valid window = `[0, headroom_bytes + min(plen, 256))`; violations ⇒ `err_window_violation`. LD zero-extends n (1–8) big-endian bytes; ST writes the low n bytes of rs big-endian.

**LDMD fields:** 0–7 = inbound SMD slots (PP `smd[i]`, zero-extended 16→64); 8 = ingress port; 9 = `md_flood` = `(2^n_ports − 1) & ~(1 << ingress)`; 10 = `hdr_present_in` as a 16-bit bitmap; 11–15 = defined zero.

**LOOKUP:** key = `rs & mask(key_width)`; linear scan of table `tbl`'s entries; hit ⇒ rd ← action (zero-extended from `action_width`), fall through; miss (or empty/unconfigured table) ⇒ rd ← 0, pc ← tgt.

**CSUMUPD hdr+off:** base = effective index; `ihl = window[base] & 0xF`; `hlen = ihl*4`; require `ihl ≥ 5` and `base+hlen` within valid window, else `err_window_violation`. Compute RFC-791 header checksum (16-bit BE words, bytes 10–11 as zero, end-around-carry fold, complement), write BE at `base+10..11`.

**SEND rs, delta:** require `−min(plen,256) < delta ≤ headroom_bytes`, else `err_send_range`. Terminates: `verdict_sent`, `egress = rs & mask(n_ports)` (empty bitmap is legal), frame = `window[headroom_bytes − delta .. headroom_bytes + plen)`. **DROP** terminates with `verdict_drop`.

**Verdicts:** 0 = sent, 1 = drop, 2 = error (error ⇒ glue drops). **Error codes:** 0 none, 1 window_violation, 2 step_budget, 3 illegal, 4 pc_range, 5 hdr_absent, 6 send_range.

**Step loop:** identical shape to `spec/model/exec.sail` — budget check, pc-range check, decode, execute; falling off imem ⇒ pc_range.

The MAP emulator context file (`ctx.txt`, one datum per line, all ints decimal or 0x-hex):

```
ingress 1
plen 60
smd 0 0x1234          # slot value   (8 lines or fewer; absent = 0)
hdr 2 1 14            # id present offset   (16 lines or fewer; absent = not-present)
table 0 48 8          # id key_width action_width
entry 0 0x02deadbeef01 0x2   # table_id key action
```

`nanuk-map-emu prog.bin pkt.bin ctx.txt` prints one JSON object:
`{"verdict": 0, "error": 0, "egress": 2, "delta": 22, "steps": 9, "frame": "aabb..."}` (`frame` hex, present iff verdict = 0).
